# Zotero Project Launcher (ZPL)

一款轻量级的 Zotero 多项目管理工具，让你像管理文件夹一样管理 Zotero 项目。

## 功能特性

- **一键新建项目**：输入名称即可创建独立的 Zotero 项目
- **项目完全隔离**：每个项目拥有独立的 data 和 profiles 目录
- **桌面快捷方式**：为常用项目生成桌面快捷方式，一键启动
- **模板管理**：支持多版本模板，自动匹配用户 Zotero 版本
- **中英文界面**：支持简体中文和 English 切换

## 安装

### 依赖
- Python 3.7+
- PySide6
- pywin32 (Windows)

### 安装步骤
```bash
pip install -r requirements.txt
python src/main.py
